macOS menu bar app

CNCjs Watchdog

Watches your CNCjs sender on grbl.local, catches the stall where Grbl sits idle while the job still says “running,” and kicks it back to life — pause, resume, keep drawing.

View on GitHub Install

What it does

  • Auto connect & reconnect — finds CNCjs whenever the Pi is up.
  • Stall recovery — pauses then resumes the sender the moment a job freezes.
  • Live % in the menu bar — 🟢 running · 🟠 recovering · ⚪ idle.
  • Notifications — connected, stall recovered, job complete.
  • Launch at login, editable config, and a rolling log.

Install

# clone & set up
git clone https://github.com/emileaublet/cncjs-watchdog.git
cd cncjs-watchdog
python3 -m venv venv && ./venv/bin/pip install -r requirements.txt

# build the menu bar app
./venv/bin/python setup.py py2app
open "dist/CNCjs Watchdog.app"

Then set your CNCjs secret in ~/.cncjs-watchdog.json — see the README.
